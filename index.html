2Dock



The script for Dash to Dock!

The 2Dock is a simple bash script to modify the dash of ubuntu 22.04 LTS to a macos style dock.
It moves all the applications to the dock and bring transparency and also hides the drive icons, All in just a single execution.
You can also go back to the default setup in this script.The code is free and opensource.
Click on the download button and you will be redirected to download page
Download the zip file , extract it and follow the instructions in the readme file.
